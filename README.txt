#PS Store

##Описание
PS Store - реализация интернет магазина комплектующих ПК на .NET 7, C#, MS SQL Server.

##Функции
-Авторизация с помощью cookie
-Работа с пользователями:
	-Создание аккаунта
	-Удаление аккаунта
	-Редактиование данных пользователя
-Работа с товарами интернет-магазина:
	-Добавление товаров в корзину
	-При добавление одинаковых товаров в корзину, товары стакаются
	-Удаление товаров из корзины
	-Изменение количества товара в корзине
	-Просмотр информации о товаре(цена, промежуточная стоимость, количество и т.п)
	-Оформление заказа
